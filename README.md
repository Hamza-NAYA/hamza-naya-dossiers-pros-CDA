## Projet TrackR

Application SaaS full-stack de gestion client pour freelances français :
authentification JWT RS256 + Magic Link, facturation conforme droit français,
paiements Stripe Connect, portail client white-label, tableau Kanban collaboratif,
154 tests PHPUnit + 23 Vitest, CI/CD GitHub Actions.

**Stack** : Symfony 7.2 · PHP 8.4 · React 19 · TypeScript 6 · PostgreSQL 17 · Redis · Docker
